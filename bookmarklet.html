🚀 실시간 오류 모니터링

📋 사용 방법

1단계: 아래 버튼을 브라우저 북마크바로 드래그하세요
2단계: http://localhost:3900 페이지에서 북마크를 클릭하세요
3단계: 페이지가 초록색 테두리로 변하면 연결 성공!
🔗 오류 모니터링 시작
✅ WebSocket 서버가 실행 중입니다 (포트 3900)

🎯 기능

  • 실시간 Hydration 오류 감지
  • 콘솔 오류 자동 수집
  • DOM 변경 감지
  • 자동 오류 수정
수동 실행 코드 (콘솔에 붙여넣기):
// 위의 북마크릿을 클릭하거나, 이 코드를 콘솔에 붙여넣으세요 console.log('🔗 WebSocket 연결 시도 중...'); const ws = new WebSocket('ws://localhost:3900'); // ... (전체 코드는 북마크릿에 포함됨)
